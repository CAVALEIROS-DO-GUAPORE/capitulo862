# Modelo de Ata em Word – Tags

O arquivo **atareuniao.docx** deve conter as tags abaixo. Ao baixar "Baixar ata em Word" numa ata publicada, o sistema substitui cada tag pelo valor preenchido no sistema.

## Tags disponíveis

| Tag | Descrição |
|-----|-----------|
| `{ataNumero}` | Número da ata em sequência do sistema |
| `{ataAno}` | Ano atual da ata |
| `{ataGestao}` | Gestão 1 ou 2 (definido pelo escrivão ao preencher) |
| `{diaAta}` | Dia da reunião (ex.: 7) |
| `{mêsAta}` ou `{mesAta}` | Mês da reunião por extenso (ex.: fevereiro) |
| `{anoAta}` | Ano da reunião |
| `{local}` | Onde foi realizada (manual ou CAPÍTULO = Augusta e Respeitável Loja Simbólica Estrela do Guaporé nº 63 na cidade de Pontes e Lacerda - MT) |
| `{demolaysPresentes}` | DeMolays presentes |
| `{senioresPresentes}` | Sêniores presentes |
| `{maçonsPresentes}` | Tios maçons presentes |
| `{mestreConselheiro}` | Nome do Mestre Conselheiro |
| `{primeiroConselheiro}` | Nome do 1º Conselheiro |
| `{segundoConselheiro}` | Nome do 2º Conselheiro |
| `{tioConselho}` | Tio escolhido como líder do conselho |
| `{horaInicio}` | Hora de início da reunião |
| `{palavraSecreta}` | Palavra secreta (opcional; apenas ritualística) |
| `{pauta}` | Tópicos discutidos (separados por vírgula) |
| `{bemdaOrdem}` | Descrição completa da reunião (campo Bem da Ordem) |
| `{horaFim}` | Horário de término |
| `{nomeEscrivao}` | Nome do escrivão |

Tags não preenchidas no sistema saem em branco no Word.
